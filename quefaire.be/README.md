# quefaire.be Scraper — Centre de Contrôle

Dashboard de contrôle pour le scraper d'événements [quefaire.be](https://www.quefaire.be).

## Fonctionnalités

- 🏙️ Scraping d'événements pour 5 villes belges (Charleroi, La Louvière, Liège, Mons, Namur)
- 🔗 Extraction des URLs d'événements avec Selenium
- 📄 Extraction des détails complets (titre, description, lieu, dates, tarif, images)
- 📊 Dashboard de visualisation des données
- 📦 Export JSON par ville
- ⏱️ Système de cache 12h
